FAQ

Video downloader questions

Which links work best?

Public video, reel, short, pin, and post links work best. Private, DRM, login-only, or region-blocked media may fail.

Can it download thumbnails?

Yes. If a public thumbnail is available, the app adds it as a separate ready asset.

Does it scrape private profile data?

No. It only shows public metadata returned by yt-dlp for the provided media URL.
